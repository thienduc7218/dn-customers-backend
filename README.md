# Denius Customers Management Application

## Requirements:

- Docker CE/Desktop 20 or above

- NodeJS LTS 16 or above

## Development

1. Docker local:

   - Use docker: `docker compose up -d`

2. Dev:

   - Install required packages: `yarn`

   - Run: `yarn start`

   - Test: `yarn test`

   - Test with coverage: `yarn test --coverage`
